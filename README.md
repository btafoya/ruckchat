# RuckChat v1 Starter

## Goals
- Single Rust server
- PostgreSQL only
- Tauri desktop
- Flutter mobile
- REST + WebSockets
- Fast deployment
- No Redis, Kafka, Elasticsearch, Kubernetes, or microservices

## MVP
1. Authentication
2. Organizations
3. Channels
4. Direct Messages
5. Realtime messaging
6. File uploads
7. Search
8. Notifications
9. Plugin SDK
10. Packaging
