# Embedded Hunspell dictionaries

These dictionaries are used by the `ruckchat-spelling` crate for server-side
spell checking.

- `en-US.aff` — based on Geoff Kuenning’s Ispell `english.aff`, distributed
  under a BSD-style license.
- `en-US.dic` — derived from the SCOWL word lists, distributed under SCOWL’s
  permissive license.

Both files come from the LibreOffice `dictionaries` repository and are
compatible with the MIT license of the RuckChat project when used as data.
