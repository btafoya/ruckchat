
# Product Requirements Document

## Vision
Build a production-ready, self-hosted collaboration platform to replace Rocket.Chat.

## Goals
- Desktop-first experience
- Native mobile apps
- Thin clients, smart server
- Single Rust backend
- PostgreSQL only
- REST + WebSockets + MCP
- Native Rust plugin SDK
- Multi-workspace
- Enterprise authentication
- Search-first UX

## Non-Goals
- Microservices
- AI features
- Built-in video/audio calls
- Workflow builders
- Plugin marketplace
- Redis, Kafka, Elasticsearch

## Functional Requirements

### Messaging
- Channels
- Private channels
- DMs
- Group DMs
- Threads
- Reactions
- Mentions
- Markdown
- Attachments

### Files
- Local/S3 storage
- File library
- Rich previews
- SHA-256 deduplication

### Authentication
- Local
- LDAP
- OpenID Connect

### Notifications
- Desktop
- Mobile push
- Email
- Notification inbox

### APIs
- OpenAPI REST
- WebSockets
- MCP server
